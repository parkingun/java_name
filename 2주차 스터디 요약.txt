2주차 (~161)
객체? = 의사나 행위가 미치는 대상

클래스? = 객체의 속성과 기능을 코드로 구현한 것

클래스명의 시작은 대문자 = 개발자들의 규칙

메서드(멤버 함수)?
멤버 변수를 사용하여 클래스 기능을 구현하는 것

패키지?
클래스 파일의 묶음

return 예약어
 함수 수행한 결과값을 반환
 함수 수행을 끝내고 프로그램 호출한 곳으로 되돌아갈 때
 
함수의 장점
1) 기능을 나누어 코드를 효율적으로 구현
2) 같은 기능의 중복 코딩 방지
3) 디버깅에 용이함

자바 이름 짓기 규약
 - 패키지 이름은 모두 소문자
 - 메서드 이름은 소문자로 시작 이름이 길어지는 경우 단어가 바뀔 때마다 대문자(가독성 향상)
  = 낙타 표기법

같은 패키지 내에서는 다른 클래스를 불러올 수 있음  
패키지가 다른 경우에는 import를 이용해서 클래스를 불러와야함.

new 예약어로 클래스 생성
클래스형 변수 이름 = new 생성자;

힙 메모리 = 동적 메모리
 - 객체가 생성될 때 사용하는 메모리 공간
 - 멤버 변수를 저장하는 공간
 
생성자
클래스를 생성할 때만 호출
생성자 이름 = 클래스의 이름

생성자 오버로드
 클래스에 생성자가 두개 이상 제공되는 경우
